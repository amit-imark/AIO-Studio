DESCRIPTION

The Forge, is the latest entry into the growing gym scene in Kuwait. We came up with name as we felt that it perfectly represents a gym in which clients are "forged" into the better version of themselves. With this name we were able to create a brand that fully embodied the nature of a forge, complete with a logo, branding, stationary, tshirts and additional elements. We also came up with the slogan "Heros are made, not born" which further developed the concept theme.
